# Диаграмма состояний (State Diagram) — «Король Лев»

## Описание

Диаграмма состояний отражает **жизненный цикл Симбы** — от рождения до восстановления трона.

### Состояния
| Состояние | Тип | Описание |
|-----------|-----|----------|
| Львенок | `<<initial>>` | Начальное состояние, Симба беззаботен и учится у Муфасы. |
| Изгнанник | `<<Bad>>` | Состояние после гибели Муфасы. Симба живёт в джунглях с принципом «Хакуна Матата». |
| Осознание | составное | Внутреннее состояние, где Симба принимает ответственность. |
| Король | `<<Final>>` | Терминальное состояние, Симба восстанавливает трон. |

### Переходы (триггеры)
| Откуда | Куда | Триггер |
|--------|------|---------|
| [*] | Львенок | `<<Рождение>>` |
| Львенок → Наследник | (внутри Львёнка) | `Муфаса учит` |
| Львенок | Изгнанник | `Гибель Муфасы / Приказ Шрама` |
| Изгнанник | Осознание | `Рафики / Воспоминание` |
| Осознание | Король | `Битва со Шрамом / Возвращение трона` |

### Составные состояния
- **Львенок** включает подсостояния: `Беззаботный` → `Наследник`.
- **Изгнанник** включает: `Стыд` → `Безразличие` (триггер: «Хакуна Матата»).
- **Осознание** включает: `ПринятиеОтветственности`.

## Код PlantUML

\`\`\`plantuml
@startuml
skinparam state {
  BackgroundColor<<Final>> #gold
  BackgroundColor<<Bad>> #lightgray
}

title Жизненный цикл Симбы

[*] --> Львенок : <<Рождение>>

state Львенок {
  [*] --> Беззаботный
  Беззаботный --> Наследник : Муфаса учит
}

state Изгнанник <<Bad>> {
  [*] --> Стыд
  Стыд --> Безразличие : "Хакуна Матата"
}

state Король <<Final>>

Львенок --> Изгнанник : Гибель Муфасы / \nПриказ Шрама

Изгнанник --> Осознание : Рафики / \nВоспоминание

state Осознание {
  [*] --> ПринятиеОтветственности
}

Осознание --> Король : Битва со Шрамом / \nВозвращение трона
@enduml
\`\`\`

## Примечания

- Состояние «Изгнанник» отмечено как `<<Bad>>` (потеря прав доступа к ресурсу PrideLands).
- Состояние «Король» — терминальное (финальное) с золотым фоном.
- Триггер «Рафики / Воспоминание» соответствует ключевому сообщению `rememberWhoYouAre` из sequence-диаграммы.
- Внутренние переходы показывают эмоциональную эволюцию персонажа внутри каждого состояния.
