# Диаграмма вариантов использования (Use Case Diagram) — «Король Лев»

## Описание

Диаграмма вариантов использования показывает, какие действия могут выполнять разные акторы в системе «Земли Прайда».

### Акторы
- **Муфаса** — законный Король.
- **Симба** — наследник, проходящий путь от львёнка до Короля.
- **Шрам** — антагонист, пытающийся захватить власть.
- **Рафики** — мудрый наставник.

### Варианты использования
| Вариант использования | Описание |
|----------------------|----------|
| Охотиться | Базовое действие Короля для поддержания баланса. Доступно Муфасе и Симбе. |
| Вернуть трон | Главная цель Симбы — восстановить справедливость. |
| Изгнать Шрама | Необходимое условие для возвращения трона (связь `<<include>>`). |
| Показать истину | Действие Рафики, которое помогает Симбе осознать себя (расширение для «Вернуть трон»). |
| Узурпировать власть | Действие Шрама по захвату Земель Прайда. |

### Связи
- **Наследование:** Симба и Муфаса — оба акторы типа Король.
- **`<<include>>`:** «Вернуть трон» включает «Изгнать Шрама» (нельзя вернуть трон, не изгнав Шрама).
- **`<<extend>>`:** «Показать истину» расширяет «Вернуть трон» (срабатывает только если Симба забыл свою судьбу).

## Код PlantUML

\`\`\`plantuml
@startuml
left to right direction

actor "Муфаса" as Mufasa
actor "Симба" as Simba
actor "Шрам" as Scar
actor "Рафики" as Rafiki

package "Земли Прайда (PrideLands)" {
  usecase "Охотиться" as Hunt
  usecase "Вернуть трон" as Restore
  usecase "Изгнать Шрама" as Exile
  usecase "Показать истину" as ShowTruth
  usecase "Узурпировать власть" as Usurp
}

Mufasa --> Hunt
Simba --> Hunt
Simba --> Restore
Simba --> Exile
Scar --> Usurp
Rafiki --> ShowTruth

Restore ..> Exile : <<include>>
ShowTruth ..> Restore : <<extend>>
@enduml
\`\`\`

## Примечания

- Вариант использования «Охотиться» доступен только законным Королям.
- «Вернуть трон» — основной сценарий Симбы, который не может быть выполнен без изгнания Шрама.
- Рафики вмешивается только при необходимости (расширение).
