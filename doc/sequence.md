# Диаграмма последовательности (Sequence Diagram) — «Король Лев»

## Описание

Диаграмма последовательности показывает пошаговое взаимодействие между объектами в ключевой сцене: **Рафики помогает Симбе вспомнить свою истинную судьбу**.

### Участники
- **Муфаса** (актор) — погибает в начале сценария.
- **Земли Прайда** (participant) — ресурс, за который идёт борьба.
- **Шрам** (актор) — антагонист, изгоняющий Симбу.
- **Симба (Львенок)** (актор) — персонаж до осознания.
- **Рафики** (актор) — наставник, отправляющий ключевое сообщение.
- **Симба (Король)** (актор) — персонаж после осознания.

### Основные сообщения
1. **Муфаса → Земли Прайда:** `править()` — начальное состояние.
2. **Шрам → Муфаса:** `предательство()` — деструктивное действие.
3. **Шрам → Симба:** `изгнать()` — переход в статус «Изгнанник».
4. **Рафики → Симба:** `показатьИстину()` — **ключевое сообщение** (rememberWhoYouAre).
5. **Рафики → Симба:** отражение в воде — визуальный ответ.
6. **Симба → Шрам:** `битва()` — кульминация.
7. **Симба → Земли Прайда:** `вернутьТрон()` — восстановление ресурса.

### Особенности
- **Деструкция объекта:** Муфаса уничтожается (`destroy M`).
- **Активация:** Рафики активирует свой метод (`activate R` / `deactivate R`).
- **Смена статуса:** Симба превращается из Львёнка в Короля.

## Код PlantUML

\`\`\`plantuml
@startuml
title Пошаговое взаимодействие: "Вспомни кто ты"

actor "Муфаса" as M
participant "Земли Прайда" as PL
actor "Шрам" as S
actor "Симба (Львенок)" as SimbaC
actor "Рафики" as R
actor "Симба (Король)" as SimbaK

M -> PL : правит()
S -> M : предательство (бросок в пропасть)
destroy M

S -> SimbaC : изгнать()
SimbaC -> SimbaC : уйти в изгнание

R -> SimbaC : показатьИстину() \n(rememberWhoYouAre)
activate R
R --> SimbaC : отражение в воде
deactivate R

SimbaC -> SimbaK : превращение
SimbaK -> S : битва()
SimbaK -> PL : вернутьТрон()

@enduml
\`\`\`

## Примечания

- Сообщение `показатьИстину()` — аналог `rememberWhoYouAre` из оригинального мультфильма.
- После изгнания Симба находится в состоянии «без прав доступа» к ресурсу PrideLands.
- Только после осознания он может выполнить `вернутьТрон()`.
