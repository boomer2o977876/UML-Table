# Диаграмма активности (Activity Diagram) — «Король Лев»

## Описание

Диаграмма активности отражает «Круг жизни» (Circle of Life) — ключевой сценарий от правления Муфасы до восстановления трона Симбы.

### Основные этапы
1. **Правление Муфасы** — начальное состояние системы.
2. **Заговор Шрама** — параллельный процесс (fork).
3. **Трагедия** — гибель Муфасы (точка ветвления).
4. **Изгнание Симбы** — переход в состояние «Изгнанник».
5. **Осознание** — встреча с Рафики, ключевое событие.
6. **Битва** — возвращение и сражение со Шрамом.
7. **Восстановление трона** — финальное состояние.

### Ветвления
- **Если Муфаса выживает?** → альтернативная история (обычно не происходит).
- **Если Симба побеждает?** → хорошая концовка (трон восстановлен).
- **Если Симба проигрывает?** → плохая концовка (Шрам остаётся у власти).

### Параллельные действия (fork/join)
- Симба растёт одновременно с тем, как Шрам вынашивает план.

### Разделы (partitions)
- **Состояние «Изгнанник»** — жизнь Симбы в джунглях с принципом «Хакуна Матата».
- **Состояние «Осознание»** — момент, когда Рафики показывает отражение.

## Код PlantUML

\`\`\`plantuml
@startuml
title Диаграмма активности: "Круг жизни"

start

:Муфаса правит Землями Прайда;

fork
  :Симба растет (Львенок);
fork again
  :Шрам вынашивает план;
end fork

:Шрам организует загон антилоп;
:Муфаса спасает Симбу;

if (Муфаса выживает?) then (нет)
  :Шрам предает Муфасу;
  :Муфаса погибает;
  
  :Шрам захватывает трон;
  :Симба изгоняется;
  
  partition "Состояние: Изгнанник" {
    :Симба живет в джунглях;
    :Принцип "Хакуна Матата";
  }
  
  partition "Состояние: Осознание" {
    :Рафики находит Симбу;
    :Рафики показывает отражение;
    :Симба осознает судьбу;
  }
  
  :Симба возвращается в PrideLands;
  :Битва со Шрамом;
  
  if (Симба побеждает?) then (да)
    :Симба восстанавливает трон;
    :Земли Прайда процветают;
    stop
  else (нет)
    :Система остается под властью Шрама;
    stop
  endif
  
else (да)
  :Альтернативная история;
  stop
endif
@enduml
\`\`\`

## Примечания

- Диаграмма отражает **бесконечный цикл жизни** — смерть старого Короля и восход нового.
- Ключевая точка принятия решения — гибель Муфасы (без этого нет развития сюжета).
- Partition «Осознание» соответствует моменту отправки сообщения `rememberWhoYouAre`.
